# Function: sum()

> **sum**(...`numbers`): `number`

Calculates the sum of all provided numbers.

## Parameters

### numbers

...`number`[]

A list of numbers to sum.

## Returns

`number`

The sum of all provided numbers. Returns `0` if no numbers are given.

## Example

```ts
sum(1, 2, 3); // 6
sum();        // 0
```
