# @petr-ptacek/js-core

Collection of small tested utils functions for **JavaScript** a **TypeScript**.

## Instalations

```shell
  npm install @petr-ptacek/js-core
```

## Usage

```ts
import { isUndefined, clamp, type Dict } from "@petr-ptacek/js-core";
```
