# object

Utilities in the **object** category.

<ul>
<li><a href="/api/object/entries">entries</a> — Returns the enumerable own property [key, value] pairs of an object with preserved key and value types.</li>
<li><a href="/api/object/get">get</a> — Safely gets a nested value from an object using a dot-separated path with full TypeScript type safety.</li>
<li><a href="/api/object/range">range</a> — Creates an array of numbers following the same semantics as Python's range.</li>
<li><a href="/api/object/shuffle">shuffle</a> — Returns a new array with elements shuffled in random order using the Fisher–Yates shuffle algorithm.</li>
<li><a href="/api/object/zip">zip</a> — Combines two arrays element-wise into pairs or applies a mapping function to each pair.</li>
</ul>
