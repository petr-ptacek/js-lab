# validation

Utilities in the **validation** category.

<ul>
<li><a href="/api/validation/isArray">isArray</a> — Checks whether the given value is an array.</li>
<li><a href="/api/validation/isBigint">isBigint</a> — Checks whether the given value is a bigint.</li>
<li><a href="/api/validation/isBoolean">isBoolean</a> — Checks whether the given value is a boolean.</li>
<li><a href="/api/validation/isDate">isDate</a> — Checks whether the given value is a Date instance.</li>
<li><a href="/api/validation/isFiniteNumber">isFiniteNumber</a> — Checks whether the given value is a finite number.</li>
<li><a href="/api/validation/isFunction">isFunction</a> — Checks whether the given value is a function.</li>
<li><a href="/api/validation/isNaNValue">isNaNValue</a> — Checks whether the given value is NaN (Not-a-Number).</li>
<li><a href="/api/validation/isNull">isNull</a> — Checks whether the given value is null.</li>
<li><a href="/api/validation/isNullable">isNullable</a> — Checks whether the given value is null or undefined.</li>
<li><a href="/api/validation/isNumber">isNumber</a> — Checks whether the given value is a number.</li>
<li><a href="/api/validation/isObject">isObject</a> — Checks whether the given value is an object (including arrays, but excluding null).</li>
<li><a href="/api/validation/isPlainObject">isPlainObject</a> — Checks whether the given value is a plain object.</li>
<li><a href="/api/validation/isPrimitive">isPrimitive</a> — Checks whether the given value is a JavaScript primitive.</li>
<li><a href="/api/validation/isString">isString</a> — Checks whether the given value is a string.</li>
<li><a href="/api/validation/isSymbol">isSymbol</a> — Checks whether the given value is a symbol.</li>
<li><a href="/api/validation/isUndefined">isUndefined</a> — Checks whether the given value is undefined.</li>
<li><a href="/api/validation/isValidDate">isValidDate</a> — Checks whether the given value is a valid Date instance.</li>
</ul>
