# @petr-ptacek/js-core

Collection of small tested utils functions for **JavaScript** and **TypeScript**.

## Installation

Install using the preferred package manager

::: code-group

```shell [npm]
npm install @petr-ptacek/js-core
```

```shell [yarn]
yarn add @petr-ptacek/js-core
```

```shell [pnpm]
pnpm add @petr-ptacek/js-core
```

```shell [bun]
bun add @petr-ptacek/js-core
```

:::

## Usage

```js
import { isUndefined } from "@petr-ptacek/js-core";

let name;

if (isUndefined(name)) {
  name = "Joe Doe";
}
```
