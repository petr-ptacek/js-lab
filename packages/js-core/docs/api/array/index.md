# array

Utilities in the **array** category.

<ul>
<li><a href="/api/array/range">range</a> — Creates an array of numbers following the same semantics as Python's range.</li>
<li><a href="/api/array/shuffle">shuffle</a> — Returns a new array with elements shuffled in random order using the Fisher–Yates shuffle algorithm.</li>
<li><a href="/api/array/zip">zip</a> — Combines two arrays element-wise into pairs or applies a mapping function to each pair.</li>
</ul>
