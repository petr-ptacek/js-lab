# browser

Utilities in the **browser** category.

<ul>
<li><a href="/api/browser/isInteractiveElement">isInteractiveElement</a> — Determines whether an element is interactive or contained within an interactive element.</li>
<li><a href="/api/browser/loadImage">loadImage</a> — Loads an image from a given source URL with optional configuration.</li>
<li><a href="/api/browser/scaleImageByAspectRatio">scaleImageByAspectRatio</a> — Scales an image to a target width or height while preserving its aspect ratio.</li>
<li><a href="/api/browser/shrinkImage">shrinkImage</a> — Shrinks an image by limiting its maximum dimensions and optionally reducing output quality.</li>
<li><a href="/api/browser/shrinkImageElement">shrinkImageElement</a> — Shrinks an image element and returns a new HTMLImageElement with the optimized result.</li>
<li><a href="/api/browser/shrinkImageFile">shrinkImageFile</a> — Shrinks an image file before upload by limiting its maximum dimensions and optionally reducing output quality.</li>
</ul>
